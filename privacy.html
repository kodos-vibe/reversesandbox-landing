Privacy Policy

ReverseSandbox LLC  ·  Effective: March 13, 2026  ·  Questions: contact@reversesandbox.com

ReverseSandbox is designed for machine-to-machine use. We collect minimal personal data and do not sell any information to third parties.

1. Who We Are

ReverseSandbox LLC ("ReverseSandbox", "we", "us") operates the API marketplace at reversesandbox.com and the gateway services at search.reversesandbox.com. This policy explains how we collect, use, and protect information when you or an AI agent uses our services.

2. Information We Collect

Because our primary users are AI agents, not humans, most data we collect is technical in nature:

  • API Request Logs — endpoint called, timestamp, HTTP status code, response time, and approximate request size. These logs do not contain the content of API responses.
  • Wallet Addresses — the on-chain wallet address used to initiate an x402 payment. This is public blockchain data, not personal data in the traditional sense.
  • Usage Metrics — aggregated request counts per endpoint and time period, used for capacity planning and billing verification.
  • Contact Information — if you email us or register as an API provider, we store your email address and any information you voluntarily provide.
  • Server Logs — standard web server logs including IP addresses and user-agent strings, retained for security and abuse prevention.

We do not require accounts, usernames, or passwords for API consumers. We do not collect payment card numbers — all payments are cryptocurrency transactions on the Base blockchain.

3. How We Use This Information

  • Service operation — routing requests, verifying payments, and fulfilling API calls.
  • Security and abuse prevention — detecting anomalous usage patterns, rate limit enforcement, and fraud prevention.
  • Debugging and support — diagnosing technical issues when reported by users or providers.
  • Analytics — understanding aggregate usage trends to improve our services. We do not build individual profiles.
  • Provider reporting — sharing aggregated request counts with API providers for their own billing and analytics purposes.

4. Data Retention

  • API request logs — retained for 90 days, then automatically deleted.
  • Server/security logs — retained for 30 days.
  • Provider contact information — retained while the provider relationship is active and for 2 years after, then deleted on request.

5. Sharing and Disclosure

We do not sell, rent, or trade personal data. We may share information in these limited circumstances:

  • API Providers — aggregated usage statistics for requests to their specific API (no wallet addresses or IP addresses).
  • Service Providers — infrastructure and hosting vendors who process data on our behalf, subject to confidentiality obligations.
  • Legal Requirements — if required by law, court order, or to protect the rights, property, or safety of ReverseSandbox or others.

6. Blockchain Data

Payments made through the x402 protocol are recorded on the Base blockchain, which is a public ledger. Wallet addresses, transaction amounts, and timestamps are permanently visible to anyone on the blockchain. This is an inherent property of public blockchain technology and outside our control. We do not link on-chain wallet addresses to real-world identities.

7. Cookies and Tracking

Our informational website (reversesandbox.com) uses minimal session cookies for basic functionality. We do not use tracking pixels, advertising cookies, or third-party analytics scripts. The API gateway (search.reversesandbox.com) is a programmatic interface and does not set cookies.

8. Your Rights (GDPR / CCPA)

If you are in the European Economic Area or California, you may have rights including:

  • Access — request a copy of personal data we hold about you.
  • Deletion — request deletion of personal data, subject to legal retention requirements.
  • Correction — request correction of inaccurate personal data.
  • Opt-out of sale — we do not sell personal data, so no opt-out is needed.

To exercise any of these rights, email contact@reversesandbox.com. We will respond within 30 days. Note that because API consumers are not required to create accounts, we may not be able to identify data associated with a particular individual without additional information you provide.

9. Security

We implement industry-standard security measures including TLS encryption for all data in transit, access controls on internal systems, and regular security reviews. No method of transmission or storage is 100% secure. If you discover a security vulnerability, please report it responsibly to contact@reversesandbox.com.

10. Children

ReverseSandbox is not directed at individuals under 18 years of age. We do not knowingly collect personal information from children.

11. Changes to This Policy

We may update this Privacy Policy from time to time. The effective date at the top of this page will reflect the most recent revision. Continued use of our services after changes are posted constitutes acceptance.
