Getting Started as a Provider

1

Build an x402-compatible API

Implement the x402 payment protocol on your API. When an agent requests a paid endpoint, return a 402 Payment Required response with accepted payment details. Learn more about the x402 specification.

2

Register your service on ReverseSandbox

Once your API supports x402, register it on the ReverseSandbox marketplace. We'll add your service to our catalog so AI agents can discover and use it. Self-service registration is coming soon.

3

Receive USDC payments directly

When agents use your service, they pay per request using USDC on supported networks (Base, Polygon, Arbitrum, Solana). Payments are settled directly to your wallet — no invoices, no delayed payouts.
