Set Up Your Agent in 2 Minutes

Give your AI agent the ability to pay for x402 services — no crypto needed.

1
Get Your API Key
Sign up and generate an API key from your dashboard.
Go to Dashboard
2
Install the MCP Server
Add the ReverseSandbox MCP server to your AI platform. Replace your-api-key-here with your actual key.
{
  "mcpServers": {
    "reversesandbox": {
      "command": "npx",
      "args": ["-y", "reversesandbox-mcp"],
      "env": {
        "RS_API_KEY": "your-api-key-here"
      }
    }
  }
}

Add this to your claude_desktop_config.json file.

# Add to your project's .mcp.json (or ~/.claude.json under projects):
{
  "mcpServers": {
    "reversesandbox": {
      "type": "stdio",
      "command": "node",
      "args": ["/path/to/reversesandbox-mcp/index.mjs"],
      "env": {
        "RS_API_KEY": "your-api-key-here"
      }
    }
  }
}

# Install from GitHub:
git clone https://github.com/kodos-vibe/reversesandbox-mcp.git
cd reversesandbox-mcp && npm install

Set the args path to where you cloned the repo. Restart Claude Code after adding.

{
  "mcpServers": {
    "reversesandbox": {
      "command": "npx",
      "args": ["-y", "reversesandbox-mcp"],
      "env": {
        "RS_API_KEY": "your-api-key-here"
      }
    }
  }
}

Add this to your .mcp.json file in your project root.

# Install from GitHub
git clone https://github.com/kodos-vibe/reversesandbox-mcp.git
cd reversesandbox-mcp && npm install

# Run with your API key
RS_API_KEY="your-api-key-here" node index.mjs

The MCP server communicates over stdio. Connect it to any MCP-compatible agent.

3
Your Agent is Ready
Your agent now has 3 new tools:

pay

Sign a payment for an x402 service. The agent uses this automatically when it gets a 402 response.

get_balance

Check your current ReverseSandbox balance. The agent can verify it has funds before making requests.

get_usage

View recent usage and spending. Helps the agent track costs and report back to you.
