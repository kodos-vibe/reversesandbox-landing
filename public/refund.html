Fulfillment & Refund Policy

REVERSE SANDBOX LLC  ·  Effective: March 13, 2026  ·  Questions: contact@reversesandbox.com

Service & Delivery Overview

Reverse Sandbox LLC provides digital API services — Web Search, Web Scraping, and Screenshot — accessible programmatically over HTTPS. All services are delivered electronically and instantly at the time of the request, simultaneous with payment.

How payment works

Usage fees are charged per API call in USD, processed via our payment infrastructure. Each API call is billed individually. The service is fulfilled — and the result returned — in the same HTTP exchange. There is no invoice cycle, no subscription, and no pre-loaded credit balance.

Current Pricing (USD equivalent)

Web Search API $0.002 USD per request
Web Scraping API $0.005 USD per request
Screenshot API $0.002 USD per request

All prices are denominated in USD. Prices may change; the price published at the time of your request is the price charged.

Refund Policy

Because API calls are digital goods delivered instantly at the time of payment, and because consumed API calls represent immediately delivered digital services, we do not issue refunds for successfully fulfilled API calls.

This is consistent with standard practice for digital content and on-demand software services under applicable consumer protection regulations.

When We Will Consider a Refund

Refunds may be issued at our discretion in the following documented failure scenarios:

  • Service unavailability — your payment was accepted but our service was down and never processed your request (i.e., you paid but received no result).
  • Systematic errors — a bug in our gateway caused incorrect or empty responses to be returned for a sequence of requests, verified by server logs.
  • Duplicate charge — you were charged more than once for the same request due to a gateway error.

Refunds are not issued for:

  • Dissatisfaction with the content of search, scrape, or screenshot results (results are determined by the live web and third-party sources).
  • Agent logic errors or misconfigured requests.
  • Requests that were successfully fulfilled but did not meet your expectations.
  • Network failures or timeouts on the consumer's end after a response was sent.

How to Request a Refund

To request a refund, email contact@reversesandbox.com within 48 hours of the affected request(s) with:

  • The payment or transaction reference for the request in question.
  • The timestamp and endpoint of the failed request.
  • A description of the error observed.

We will review and respond within 3 business days. If a refund is approved, it will be issued in USD to the original payment method.

Cancellation Policy

ReverseSandbox operates on a pay-per-use model with no subscriptions. There is nothing to cancel. You stop using the service by simply not making further requests — no notice or cancellation process required, and no recurring charges will occur.

If you are a registered API provider and wish to delist your API from the marketplace, email contact@reversesandbox.com and we will remove your listing within 2 business days.

Payment Currency

All transactions are denominated and settled in USD (US Dollars). Usage fees are charged per API call at the published rates. Payment processing is handled securely by our payment infrastructure. No payment card data is ever collected or stored on our servers.
